# NewsChallenge
Browse Top Stories - View the most popular stories on HackerNews

Browse New Stories - Stay updated with the latest submissions

Material Design - Clean, modern UI with Angular Material components

Responsive Design - Works seamlessly on mobile, tablet, and desktop

Fully Accessible - WCAG compliant with ARIA labels and keyboard navigation.

Zoneless Architecture - Built with Angular's latest zoneless change detection

Lazy Loading - Stories load only when needed for better performance

Grid Layout - Two-column card layout for optimal content viewing

Direct Links - Quick access to original articles

Real-time Updates - Live data from HackerNews Firebase API

# TECH STACK
Framework - Angular 20
UI Library - Angular Material
Language - TypeScript 5.0+
API - HackerNews Firebase API
Architecture: Standalone Components, Zoneless
Angular HttpClient with RxJS
Build Tool: Angular CLI

# Prerequisites
Ensure you have the following installed:

-Node.js: v18.x or higher

-npm: v9.x or higher

-Angular CLI: v20.x or higher

# Installation

1. Clone the repository
Run this command on bash = git clone https://github.com/dexter0001/news-challenge.git

2. Install dependencies, Run this command on bash = npm install

3. Install Angular Material, Run this command on bash = ng add @angular/material

4. Running the Application Development Server, Run this command on bash = ng serve

Navigate to http://localhost:4200/. The application will automatically reload if you change any of the source files.

# Run Tests:

Run this command on bash = ng test

# Run Tests with coverage

Run this command on bash = ng test --code-coverage
